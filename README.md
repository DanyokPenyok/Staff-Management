## ✨ Возможности

* **Полный цикл CRUD:** Добавление, редактирование, удаление и просмотр записей о сотрудниках.
* **Работа с файлами:** Сохранение базы данных в текстовые файлы и загрузка из них.
* **Слияние баз:** Возможность объединить (Merge) текущую таблицу с данными из другого файла.
* **Умный поиск:** Фильтрация сотрудников по фамилии, должности или отделу в реальном времени.
* **Контроль изменений:** Система предупредит о необходимости сохранения перед закрытием программы или созданием новой базы.

## 🛠 Технические особенности

В проекте реализованы ключевые принципы ООП и прикладного программирования:
* **Custom Vector:** Вместо `std::vector` реализован собственный шаблонный динамический массив `Vector<T>` с поддержкой итераторов.
* **Model/View Architecture:** Использование `QAbstractTableModel` для эффективного отображения и управления данными в `QTableView`.
* **Serialization:** Реализована потоковая запись и чтение данных через `QTextStream` и `QDataStream`.
* **Modern UI:** Интерфейс включает в себя главное меню, панель инструментов (Toolbar) с иконками и строку состояния (Status Bar).
